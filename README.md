# Budget Manager

## Overview
Web Based Stock Market + Options Market Application with terminal; interface 

## Features
- Check Stock price
- Check Options Price
- Get Options Max Profit + Max Loss
- Margin porfit Calculator 
- Get data from the Federal Resevere
